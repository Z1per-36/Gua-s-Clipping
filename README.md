# 新聞追蹤小幫手 (News Clipping Tool)

這是一款專為現代人設計的輕量級、在背景自動運行的「每日新聞報表產出器」。只需輕鬆設定一次，它就會在您指定的時間，自動為您抓取各方來源（如 Google News, Reddit 等）的最新新聞，過濾時效，並彈出一份專屬的網頁報表！

## 🌟 核心特色
- **🤖 全自動排程運作**：設定時間一到，自動彈出最新新聞結果。
- **🌍 支援跨平台**：支援 Windows 及 MacOS，提供系統常駐列背景服務。
- **📅 視覺化日曆與過濾**：內建自訂日曆組件，完美過濾年代久遠的舊新聞。
- **🚫 防止重複開啟 (IPC Lock)**：如果背景已在執行，雙擊軟體會自動彈出您的設定介面，防止多開干擾。

## 📥 安裝與執行 (首次啟動必看指南)
由於本軟體為完全免費的開源專案（無購買企業數位簽章），當您首次下載執行時，作業系統的安全機制可能會跳出「未知開發者」的警告。這屬於正常現象，請完全放心！您可以依照以下步驟解鎖並執行：

### 🍎 MacOS 玩家
1. 下載後，在 Finder 中找到 `NewsClippingTool`。
2. **請不要直接快按兩下**！請按住鍵盤上的 `Control` 鍵，然後用滑鼠「點擊應用程式圖示」，接著在選單中選擇**「打開」**。
3. 系統會跳出警告視窗，此時會多出一個**「打開」(Open)** 的選項，點擊它即可順利執行！（這個動作只需做一次，之後就可以正常雙擊打開了）。
   *(若仍然無法打開，請至「系統設定」>「隱私權與安全性」> 往下拉找到此軟體，點擊「強制打開」)*

### 🪟 Windows 玩家
1. 雙擊下載回來的 `NewsClippingTool.exe`。
2. 如果畫面上跳出深藍色的「Windows 已保護您的電腦」畫面。
3. 請點擊畫面左側的 **「其他資訊」 (More info)**。
4. 接著點擊右下角出現的 **「仍要執行」 (Run anyway)** 即可順利啟動！
