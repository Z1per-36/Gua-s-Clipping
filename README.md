# Gua's Clipping

Gua's clipping 是一個在背景自動運行的跨平台新聞報表生成工具。使用者可設定關鍵字、新聞來源與擷取時間，系統將於後台自動抓取並整合新聞資料，產出過濾時效後的 HTML 報表。

## 系統特性
- 自動化排程器：支援使用者定義每天特定的發送時間。
- 跨平台支援：相容 Windows 與 MacOS 作業系統，提供系統背景常駐服務。
- 時間範圍過濾：內建日曆模組，剃除不符時效的新聞內容。
- 單一執行個體限制 (Single Instance)：利用 IPC Socket 通訊，防止本地發生多重執行個體衝突。當偵測到重複啟動事件時，將自動喚醒既有服務的設定介面。

## 作業系統安全性設定說明
由於此開源專案發布時未隨附商業組織之數位簽章 (Code Signing Certificate)，Windows 與 MacOS 的安全性機制將於首次執行時觸發警告。此為系統標準行為，請參照以下指示放行程式執行。

### MacOS 
1. 透過 Finder 導航至已下載之 `NewsClippingTool` 二進位檔。
2. 按住鍵盤 `Control` 鍵並以滑鼠點選該檔案，於快捷選單中選擇「打開」。
3. 在系統顯示的安全性警告對話框中，點擊「打開」(Open) 按鈕。
   （若上述步驟無效，請進入「系統設定」>「隱私權與安全性」，於畫面下方尋找攔截紀錄，並點擊「強制打開」）

### Windows
1. 啟動 `NewsClippingTool_windows-latest.exe`。
2. 當 Windows SmartScreen 視窗阻擋執行時，點擊對話框內左下角之「其他資訊」(More info)。
3. 點擊「仍要執行」(Run anyway)。
